C2M ADMIN Application

As part of my work at Infosys Ltd. I developed a Concept to Market Admin application for Estee Lauder Companies. C2M Admin is a web application designed to be used by the internal business users to manage their product life cycle from Concept till it was launced as a product in Market. My role in the project was a backend developer. I participated in all phases of system development life cycle, from requirements analysis through system implementation. The project used Agile methodology for its development. Tech stack used for backend development included Spring Boot Microservices, GraphQL, Redis, JUnit, Mockito, Apollo platform, AuthFilter, Maven, Kibana,Elastic Search, Logstash, Bitbucket, JIRA, Jenkins, SonarQube, Spring Tool Suite, Oracle SQL Developer, Microsoft Azure.

This was a migration project. The legacy application was designed in Visual Basic. It required the analysis of VB code and stored procedures and then design an equivalent Java code using Spring Boot. It was implemented in a team of 5 members. 2 frontend developers, 2 backend developers and a team lead.
